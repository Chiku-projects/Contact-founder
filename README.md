🚀 Advanced Website Contact Scraper

A professional Selenium-based contact scraping tool that extracts business emails and phone numbers from websites.

Built for freelancers, lead generation, marketing agencies, and automation projects.

📌 Features

✅ Extracts real business emails
✅ Extracts valid phone numbers (strict filtering)
✅ Removes false positives (versions, dates, junk text)
✅ Filters free email domains (optional)
✅ Scrolls page to load dynamic content
✅ Saves output to CSV & Excel
✅ Handles multiple websites from input file
✅ Headless browser support
✅ Clean structured output
 🛠 Technologies Used

.Python 3.12

.Selenium

.Pandas

.Regex

.Chrome WebDriver
